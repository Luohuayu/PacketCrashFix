# PacketCrashFix
MOD发包崩服修复

版本: 1.12.2

修复各种MOD缺少校验导致的发包生成区块等崩服方法

### 使用方法
正确安装方法:
1. 将mods/ClassFixer.jar安装到服务端mods目录
2. 在服务端目录新建ClassFixer目录, 然后在本项目的ClassFixer里下载需要的修复类

无脑安装方法: 下载本项目, 解压并删除掉README.md和src, 复制到服务端根目录
